# Python Image Merge

## Require

- Install Python 3.7 or higher
- Install FastAPI https://fastapi.tiangolo.com/ko/
- Install Command 

```
python3 -m pip install --upgrade Pillow
python3 -m pip install --upgrade python-multipar
```

## Test

Start Server 

```
$ bash run.sh
```
